# 开放性词汇目标检测

## 一、基础知识点

1、开放性词汇：指模型能够预测预先定义好的类别之外的类别

2、弱监督信号：无需人工精细标注、仅提供粗略关联信息或已有基础能力的引导数据，比如：

​	（1）预训练视觉语言模型的跨模态能力（如：CLIP）：经海量图像-文本对训练，匹配图像和文本的特征，避免		从零开始构建图文关联的高额成本。

​	（2）图像文本配对数据：只有图像以及对该张图像内容描述的文字组合，没有精准的物体位置标注信息。

​	（3）图像级标签

​	（4）模型生成的伪标签（自监督学习）

3、视觉定位：将语义概念与图像区域关联起来，可分为以下两种

​	（1）短语定位：将句子中的所有名词与图像区域关联起来

​	（2）指标表达理解：仅将句子中描述的对象与图像区域关联起来

4、弱监督检测与分割：

​		只有图像级标签没有边界框或者掩码

5、开放集检测与分割：

​		要求对已知类别进行分类，并识别单个”未知“类别，但无需对未知类别进行分类

6、开放世界检测与分割：

​		简单来说，就是在某一时刻出现一个新类别，然后将这个新的类别作为新的已知类别加入到训练集中，在下一时刻出现另外一个新的类别继续加入到训练集中进行训练，循环进行，未知类别会越来越少。

7、分布外检测（OOD Detection）：

​		测试数据和训练数据不是同一个分布，有分布偏移，分布偏移分为两类：

​		（1）语义偏移：分布外类别与训练类别完全不一样

​		（2）协变量偏移：分布外样本与训练类别一致，但是分布不同（可能是草图或者是对抗样本）

8、大型视觉-语言模型（VLMs）

​	（1）图文对比学习预训练（CLIP）

​	（2）结合局部于一的自监督学习（DINO）

​	（3）文本到图像（T2I）扩散模型

​	（4）参数高效微调（PEFT）
